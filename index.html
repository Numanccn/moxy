Welcome to Binder

Binder is an infographic project hosted on GitHub Pages, designed to explore and implement responsive web layouts using Flexbox and Grid. Over time, additional functionalities will be introduced to experiment with various web development techniques and enhance user interaction.

Beyond the techy stuff, this project is mainly for the daily tasks of employees at Moxy Hotel in Kelsterbach, where I currently work as a Front Office Associate. Binder is an all-in-one platform for work-related info, bringing multiple sources together in one place. This makes gathering the right details quick and easy when you need to make a decision or solve a problem. Plus, it includes handy templates for daily emails and other written communications to help your workflow.
